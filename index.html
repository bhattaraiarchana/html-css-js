👨‍🎨 About the Developer

I'm a passionate frontend learner with big dreams! 💻🔥

It all started with a single <div>

Then I learned why semantics matter more than just making it work.

“Semantics is about using the right elements for the right job.” — MDN

Here’s a tiny snippet I loved:

<header>
  <h1>My Site</h1>
  <nav aria-label="Primary">...</nav>
</header>

Keyboard tip: press Tab to focus links, Enter to activate. Sample output often looks like Build succeeded.
